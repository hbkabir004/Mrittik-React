Welcome


Mrittik-React is a feature-filled React app and component library that ships with hundreds of ready-to-use components for construction, architecture, or builders Companies. The limitless possibilities come in your hand with its rich component library. Mrittik-React is a simple and clean design. All files are organized. We believe it will be easy to use and edit them. The theme features a beautiful Shop Page with Shop Page with Ecommerce functionality, many ready-to-use hand-crafted components, widgets, pages, and so on. The cool thing is that you can use these components on any page with the utmost possible ease. The theme fits any device just because of its improved responsiveness. This powerful and amazingly flexible theme suits the best for many web applications.

Installation

Welcome to React! React helps you build modern applications for the web, mobile, or desktop.

For getting started an React application you needs two things as Prerequisites.

Prerequisites


Before you begin, make sure your development environment includes Node and an npm package manager.

Node.js


Download latest version of node.js from nodejs.org.

Install Node.js using downloaded file

To check your node version, run node -v in a terminal/console window.

Yarn package manager


React CLI, and React apps depend on features and functionality provided by libraries that are available as yarn packages. To download and install yarn packages, you must have an yarn package manager. This Quick Start uses the yarn client command line interface, which is need install by using npm, npm default with nodejs. To install yarn, run npm install -g yarn To check that you have the yarn client installed, run yarn -v in a terminal/console window.

For better understanding we suggest you to once go through official documentation of React from Reactjs.org

Downloaded Files


After extract the main zip file, go to Mrittik folder and you will get three separate project folder there. Use one which fits your needs.

Installation


  • For starter we need to setup node js, go to https://nodejs.org/en/download/ & download appropriate version for your os.
  • After download setup nodejs
  • Run npm install -g yarn, wait a bit
  • Run yarn install, wait a bit
  • Now run yarn start, it should open up http://localhost:3000 on your browser

Change font

Simplest way to change font is to go to google fonts, select fonts, click embed copy the link. Now go to public/index.html and paste the link in head tag. Please remove the links of any unused fonts from the head.

Routing

Mrittik use react-router as base routing system. To add a new route in the template, you can follow the steps below.

Create new route

We place all our route in src/Routes/Router.js file.In this folder is the entry point of all pages, you can basically set your pages in this file.

Refer below mentioned code to create a route: 

import { createBrowserRouter } from "react-router-dom";
import { productsAndCartData } from "../components/forJSON/getCart&ProductsData";
import ShopCategory from "../containers/ShopCategory";
import Main from "../Layout/Main";
import NewPage from "../Pages/NewPage";

export const routes = createBrowserRouter([
  {
    path: "/",
    loader: productsAndCartData,
    element: <Main/>,
    children: [
        {
            path: '/YOUR-PATH',
            element: <NewPage/>,
        }

]);
